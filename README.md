## 📌 Project Overview

This project is a Java-based Gadget Shop system developed for coursework.  
The application allows users to create and manage different types of gadgets using a graphical user interface (GUI).

The system demonstrates key Object-Oriented Programming (OOP) concepts including inheritance, polymorphism, encapsulation, and abstraction.

---

## ⚙️ Features

- Add a Mobile phone
- Add an MP3 player
- Store gadgets using an ArrayList
- Display all gadgets
- Make a phone call (Mobile only)
- Download music (MP3 only)
- Input validation using try/catch
- GUI built using Java Swing

---

## 🧠 OOP Concepts Used

### 1. Inheritance
- Mobile and MP3 classes extend the Gadget superclass.

### 2. Encapsulation
- All attributes are private and accessed using getter methods.

### 3. Polymorphism
- ArrayList<Gadget> stores different object types (Mobile and MP3).

### 4. Method Overriding
- The display() method is overridden in subclasses.
